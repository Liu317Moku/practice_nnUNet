<<粗淺步驟提醒>>
從conda.ipynb 開始執行，切記在執行前所有程式碼前需要先將帳號replace成自己的，再開始執行
從conda開始由上往下依序執行，看到 [數字] 表示完成；[*]表示正在執行，若還在執行請不要將畫面重跑，請耐心等待

執行順序  git clone->conda.ipnb->pip.ipnb
執行過程中請記得要看每個shell中的備註，以減少執行時發生的錯誤

執行到conda.ipnb中這行
!/opt/conda/condabin/conda run -p /home/glolg/.conda/envs/nnunet python -m ipykernel install --user --name nnunet --display-name "Python (nnU-Net)"
執行完請記得將整個網頁refash一次再繼續執行其他程式

開始訓練前，請至terminal進入環境conda activate nnunet，並執行後續安裝任務

-PyTorch 2.x 的 Inductor 後端在做 torch.compile (即 TorchDynamo) 時需要用 C 編譯器 (gcc 或 clang)，但你的環境沒有安裝或找不到 C 編譯器，用 Conda 安裝 GCC/G++  -

conda install -y -c conda-forge graphviz
conda install -y -c conda-forge gcc_linux-64 gxx_linux-64

-指定環境變數-

export CC=$CONDA_PREFIX/bin/x86_64-conda-linux-gnu-gcc
export CXX=$CONDA_PREFIX/bin/x86_64-conda-linux-gnu-g++

此外預設課程中的gpu可使用有7個，編碼為0~6，執行前也請修改為只用的gpu代碼
